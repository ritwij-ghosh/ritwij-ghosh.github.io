Winston AI: Agentic Travel Butler

Overview

Winston AI is a chat-first mobile travel concierge built with Expo/React Native, designed to handle end-to-end trip planning and booking autonomously. The user describes a trip in natural language — destination, dates, budget, preferences — and Winston handles everything from itinerary generation to flight search, hotel booking, and calendar export.

Agentic Architecture

The core of Winston is an agentic orchestration layer built around 8 production MCP (Model Context Protocol) servers, each responsible for a distinct domain: flight search, hotel availability, restaurant discovery, activity curation, weather, calendar export, currency conversion, and itinerary persistence. A custom orchestration gateway routes LLM-generated tool calls to the appropriate MCP server, collects responses, and feeds them back into context — enabling fully autonomous multi-step pipelines with no human-in-the-loop.

The gateway enforces a strict normalized itinerary schema across all boundaries: LLM outputs are validated against the schema before persistence, FastAPI routes accept and return typed Pydantic models matching the schema, and Supabase/Postgres tables mirror the same structure. This ensures the itinerary state is always consistent regardless of which server updated it.

Backend & Data Layer

The FastAPI backend exposes REST endpoints consumed by the mobile client and the MCP gateway. Authentication and row-level security are handled through Supabase. Real-time itinerary updates use Supabase subscriptions, so the mobile UI reflects changes as they're written by the agent without manual polling.

Key Engineering Decisions

Choosing MCP as the tool-calling protocol rather than raw function calling allowed each integration to be independently versioned and swapped without touching the orchestration layer. The orchestration gateway also implements retry logic and graceful degradation — if one MCP server is unavailable, the agent continues planning with the remaining tools and flags the missing data to the user.
